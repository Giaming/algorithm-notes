- 回溯函数终止条件

回溯算法代码模板：
```java
public void backtrack(原始参数) {
    if(终止条件) {
        // 一些逻辑操作
        return;
    }
    for (int i = for循环开始的参数; i < for循环结束的参数; i++) {
        // 一些逻辑操作
        
        // 做出选择
        
        // 递归
        backtrack(新的参数)
        
        // 一些逻辑操作
        
        // 撤销选择
    }
}
```
